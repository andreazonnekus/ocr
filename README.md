# Welcome to my Support Tools
What started as a project for handwriting OCR, now also contains a variety of models and utilities I've been building for university and my personal projects. It is a CLI based tool which allows the user to call specific classes and functions dynamically imported from the `classes/` folder

It uses a variety of libraries to implement solutions including:
+ Keras
+ OpenCV
+ NLTK
+ Sklearn
+ Matplotlib
+ Seaborn
